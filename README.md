# MicroX Operating System

MicroX — учебная операционная система для архитектуры x86.  
Цель проекта — изучение низкоуровневого программирования, написание ядра и драйверов, создание загрузочного ISO и запуск в QEMU.

## Зависимости
- NASM
- i386-elf-gcc / i386-elf-ld (кросс-компилятор)
- GRUB (grub-mkrescue)
- QEMU
