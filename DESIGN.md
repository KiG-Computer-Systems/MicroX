# Архитектура MicroX

## Цели
- Минимальный загрузчик на NASM
- Ядро в формате ELF
- Поддержка прерываний (IDT)
- Драйверы ввода (клавиатура, мышь)
- Простая система вывода текста

## Компоненты
- **boot.asm** — загрузчик
- **kernel.c** — точка входа ядра
- **interrupts.c / interrupts.asm** — настройка IDT, обработчики IRQ
- **io.c** — функции `inportb` и `outportb`
- **print.c** — вывод текста
- **mouse.c** — драйвер мыши
